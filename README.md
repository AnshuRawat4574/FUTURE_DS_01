# Project Overview

## Task 1 – Superstore Sales Dashboard

### Objective

Analyze Superstore sales data and create an interactive dashboard to understand sales and profit performance.

### Key Insights

* Technology category generated the highest sales.
* West region contributed the highest revenue.
* Copiers generated the highest profit.
* Tables generated losses.

### Files

* Sample-Superstore.csv
* Task1_Dashboard.pbix

## Task 2 – Customer Retention & Churn Analysis

### Objective

Analyze customer churn behavior and identify factors affecting customer retention.

### Key Insights

* Month-to-month customers showed the highest churn.
* Fiber optic users had higher churn rates.
* Customer tenure strongly impacted retention.
* Long-term contracts improved retention.

### Files

* Task2_Churn_Dashboard.pbix
* Task2_Dashboard.png

---

## Task 3 – Marketing Funnel & Conversion Performance Analysis

### Objective

Analyze marketing campaign performance and identify opportunities to improve customer conversion rates.

### Key Insights

* Secondary and tertiary education groups achieved higher conversions.
* Cellular contact method performed best.
* Management and technician professions showed strong conversion rates.
* May and August recorded the highest campaign performance.

### Files

* bank-full.csv
* Task3_Marketing_Funnel.pbix
* Task3_Dashboard.png
