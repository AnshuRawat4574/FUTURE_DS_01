# Task 1 - Superstore Sales Dashboard

## Objective
Analyze Superstore sales data and create an interactive Power BI dashboard.

## Tools Used
- Power BI
- CSV Dataset

## KPIs
- Total Sales: 2.30M
- Total Profit: 286.40K

## Visualizations
1. Sales by Category
2. Profit by Sub-Category
3. Sales by Region
4. KPI Cards

## Insights
- Technology category has highest sales.
- West region contributes highest sales.
- Copiers generate highest profit.
- Tables generate losses.
